# maofan-apn

APN service for Maofan

## POST /notifier/on

**Request parameters**

| Name | Description |
| :-: | :-- |
| device_token | Device token from app as a string |
| oauth_token | Fanfou OAuth token |
| oauth_token | Fanfou OAuth token secret |

## POST /notifier/off

**Request parameters**

| Name | Description |
| :-: | :-- |
| device_token | Device token from app as a string |
| oauth_token | Fanfou OAuth token |
| oauth_token | Fanfou OAuth token secret |

## GET /notifier/check

**Request parameters**

| Name | Description |
| :-: | :-- |
| device_token | Device token from app as a string |
| oauth_token | Fanfou OAuth token |
